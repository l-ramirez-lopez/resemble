# A fast distance algorithm for two matrices written in C++

Computes distances between two data matrices using "euclid", "cor",
"cosine"

## Usage

``` r
fast_diss(X, Y, method)
```

## Arguments

- X:

  a matrix

- Y:

  a matrix

- method:

  a `string` with possible values "euclid", "cor", "cosine"

## Value

a distance matrix
